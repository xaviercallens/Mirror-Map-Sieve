# Alien Mathematics: R(3,3,4) and Beyond

**Final Status: Archive Mode (June 2026)**

This repository contains the infrastructure and results for the "Alien Mathematics" investigation conducted by the SocrateAI Agora multi-agent system. The goal was to verify claims generated by high-performance LLMs (e.g., AlphaEvolve) concerning Ramsey bounds and tensor rank on discrete hypercubes.

## Key Outcomes

1. **No "Alien" Mathematics**: All tested claims that contradicted known literature were proven false via computational falsification or literature review. The LLMs suffered from severe hallucination and "wishful thinking" biases.
2. **High-Performance Rust Infrastructure**: We built a heavily optimized Rust-based Simulated Annealing solver capable of **2.5 to 21 million steps per second**. This represents a 420,000x speedup over Python implementations. 
3. **The Claim Verification Pipeline**: We developed a robust 4-gate verification standard:
   - Gate 1: Literature Grounding
   - Gate 2: Computational Falsification (Rust SA)
   - Gate 3: Lean 4 Formalization (0 sorry, 0 axiom)
   - Gate 4: Peer-Style Critical Review
4. **ExactRationalWitness**: A genuinely publishable Lean 4 contribution mapping hypercube bounds onto the exact rational field $\mathbb{Q}$, shifting verification from complex analysis to computable $\mathbb{Q}$-decidability.
